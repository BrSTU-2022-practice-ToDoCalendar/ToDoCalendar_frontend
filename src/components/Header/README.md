# Header

**Header** - компонент, который возвращает заголовок.

Нужен для того, чтобы на любой странице выглядил одинакого.

```html
<header>
  <div class="title">...</div>
  <div class="nav_buttons">...</div>
</header>
```

Принимает параметр:

- title: String
- Icons: JSX список

Пример:

```jsx
<Header title="Tassker" />
```
